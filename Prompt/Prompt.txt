Project Overview

Create a modern, elegant, premium-quality, and fully responsive café e-commerce website for "Velora Café" — a cozy artisan café brand that sells handcrafted coffees, teas, cakes, bakery items, desserts, healthy bowls, and soft drinks.

Tagline:
“Where Every Sip Feels Love.”

The website should feel warm, cinematic, cozy, and premium with a handcrafted café atmosphere. The design must combine natural café aesthetics with modern luxury UI/UX.

The website must be 100% mobile responsive and visually balanced across phones, tablets, laptops, and desktops. Mobile layout should NEVER feel compressed, overcrowded, or like all elements are forced into one line.
